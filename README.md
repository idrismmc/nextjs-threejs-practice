# Suburbia Skateboards - Demo Website

This project is showcases as part of learning exercise for Next.js and Three.js.

## Tech Stack

- Next.js
- Tailwind CSS
- Three.js
- React
- React-Three-Fiber
- React-Three-Drei

## Setup

```bash
npm install
npm run dev
```

The demo is hosted on [Vercel](https://nextjs-threejs-practice.vercel.app/)
